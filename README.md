## About
This repository contains all solved lab reports of **Data Communications and Computer Networks (CSC339)**.  
The labs cover Ethernet cabling, Cisco Packet Tracer, network topologies, OSI & TCP/IP models, and basic LAN implementation.

---

## Labs Included
| Lab No. | Title |
|---------|-------|
| Lab 01  | Ethernet Cabling and Twisted Pair Cable Construction |
| Lab 02  | Introduction to Packet Tracer and Network Topologies |
| Lab 03  | Implementing Different Network Topologies in Cisco Packet Tracer |
| Lab 04  | OSI Model, TCP/IP Model and Basic LAN Implementation |
| Lab 05  | Create a Simple Network Using Packet Tracer |
| Lab 06+ | Additional labs from the Lab Manual |

---

## Tools Used
- Cisco Packet Tracer  
- Ethernet Cabling Tools (Crimper, Cable Tester, RJ45 connectors)  
- Twisted Pair Cables (CAT5/CAT6)
